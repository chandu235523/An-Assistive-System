### Features:

* Blind Mode – Detects obstacles and provides audio warnings.
* Deaf Mode – Detects environmental sounds (e.g., horn, siren, help) and displays alerts visually.
* Mute Mode – Converts text into speech for users who cannot vocalize.
* GUI with integrated controls.
* Optional web-based interface.
* Real-time system feedback.
* Easy-to-use interaction buttons.
* Optional logo \& branding integration.
